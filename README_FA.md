# 🇮🇷 فارسی

## 🚀 معرفی

به این مخزن خوش آمدید.

در این پروژه نحوه راه‌اندازی **Tor Browser** با استفاده از **Snowflake Bridge** و استفاده از **SOCKS5 Proxy** آن در **v2rayN** به‌صورت کامل آموزش داده شده است.

هدف این آموزش، آشنایی با نحوه کار Tor، پیکربندی صحیح مرورگر Tor و استفاده از پروکسی محلی آن در نرم‌افزارهایی است که از SOCKS5 پشتیبانی می‌کنند.

تمام مراحل همراه با ویدئو، تصاویر و راهنمای رفع مشکلات ارائه شده است.

## 🛠 این پروژه چه کاری انجام می‌دهد؟

در این آموزش یاد می‌گیرید چگونه:

- Tor Browser را نصب کنید.
- Snowflake Bridge را فعال کنید.
- به شبکه Tor متصل شوید.
- از پروکسی محلی SOCKS5 مرورگر Tor استفاده کنید.
- v2rayN را برای استفاده از این پروکسی پیکربندی کنید.
- در صورت نیاز، TUN Mode را برای عبور ترافیک سازگار از طریق تنظیمات فعال کنید.

---

## 🌍 نحوه عملکرد

پس از اتصال موفق Tor Browser به شبکه Tor با استفاده از Snowflake Bridge، یک پروکسی محلی SOCKS5 روی سیستم ایجاد می‌شود.

سپس v2rayN به این پروکسی متصل شده و برنامه‌هایی که از آن استفاده می‌کنند، ترافیک خود را از طریق Tor Browser ارسال می‌کنند.

```
برنامه
   │
   ▼
v2rayN
   │
   ▼
127.0.0.1:9150
   │
   ▼
Tor Browser
   │
   ▼
Snowflake Bridge
   │
   ▼
Tor Network
   │
   ▼
وب‌سایت مقصد
```
